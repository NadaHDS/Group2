Welcome to the Hospital Database System

A comprehensive solution for managing hospital data efficiently.

About the Project

This project delivers a robust hospital database system built with MySQL and Microsoft Access. It is designed to simplify the storage and management of vital hospital information, including patient records, prescriptions, laboratory tests, medications, and appointments.

Main Features

  • Structured Data Management: Organized tables for Patients, Prescriptions, Diagnoses, Appointments, Lab Tests, and Medications.
  • Smart SQL Queries: Efficient retrieval of patient prescriptions, appointment summaries, disease-specific medication data, and lab test reports.
  • Integrated Data Sources: Combination of real-world hospital data and generated sample data for testing.

How to Get Started

  1. Import the provided .sql file into your MySQL environment.
  2. Open the .accdb file in Microsoft Access to explore the database.
  3. Run the SQL scripts to interact with and analyze the dataset.

Project Goals

The hospital database system aims to:

  • Streamline hospital operations and data handling.
  • Generate insightful reports for better healthcare decisions.
  • Provide quick access to patient information, prescriptions, lab tests, and appointments.
